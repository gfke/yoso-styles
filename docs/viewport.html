Viewport

Viewport is a predefined flex model definition for horizontal, vertical or centralized layout

Example:
viewport row

Non flex item
Flex item

HTML source

    
<div class="example viewport--row">
    <div class="viewport__item content--pad">
        Non flex item
    </div>
    <div class="viewport__item--flex content--pad">
        Flex item
    </div>
</div>
