Helper

Helpers are CSS classes helping you add additional settings to your layout.

  • Visibility
  • Block alignments
  • Content layout

Visibility

Class name Description
invisible The item / block is not visible but invisible elements take up space on the page.
hidden The item / block is not visible and do not take up space on the page.

Block alignments

Class name Description
float-left Float the block to left.
float-right Float the block to right.

Content layout

Class name Description
content--pad Set default padding on all sides. Top, bottom, left and right.
content--pad-vertical Set default padding on top and bottom.
content--pad-horizontal Set default padding on left and right.
content--max-size Set the width to defined max size.
content--center Set margin to auto on left and right and the width to defined max size. If the page is wider than the maximum width of the block, the block is being centered on the page.
