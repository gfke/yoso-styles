Yoso Styles

Yoso Styles is simplified css boilerplate for developing web application base on javascript framework like Angular or Ember.
Unlike other css boilerplates as Twitter Bootstrap or Foundation it do not have any predefined task that require JavaScript. So there are no predefiend jQuery plugins or Angular directives. It is pure CSS / SASS!

Yoso Styles will not do all the css work for you. It deliver basic styling / boilerplate you have to extend on your application demands.

Viewport

Viewport is a predefined flex model definition for horizontal, vertical or centralized layout
